# Get Started!

Go to the server folder by using terminal type cd server

```shell
npx hardhat install
npx hardhat compile
npx hardhat run scripts/deploy.js --network sepolia
```

Now go to the client folder by using new terminal type cd client

```shell
npm install
npm run
npm start
```
